# Инструкция по запуску Telegram Mini App

## 🚀 Быстрый старт

### Шаг 1: Деплой на Vercel

1. **Перейдите на [vercel.com](https://vercel.com)**
2. **Нажмите "New Project"**
3. **Выберите репозиторий**: `Donyado/telegram-miniapp-demo`
4. **Нажмите "Create Project"**

### Шаг 2: Настройка переменных окружения

В панели Vercel:
1. Перейдите в Settings → Environment Variables
2. Добавьте переменные:
   ```
   BOT_TOKEN = 8391693320:AAGfEICnRg12DZFxdRB_qVXW-d4st71R58A
   NODE_ENV = production
   ```

### Шаг 3: Настройка Telegram бота

1. **В Telegram найдите @BotFather**
2. **Команда `/setdomain`**
   - Выберите бота: `@testappersmini_bot`
   - Укажите домен: `ваш-проект.vercel.app`
3. **Команда `/newapp`**
   - Выберите бота
   - Имя приложения: `MiniApp Demo`
   - URL: `https://ваш-проект.vercel.app`

## 🎯 Как это работает

### Для пользователей

1. **Пользователь находит бота** в Telegram
2. **Выбирает сферу бизнеса** (рестораны, магазины, услуги и т.д.)
3. **Оставляет номер телефона** для связи
4. **Получает доступ** к интерактивному шаблону мини-приложения
5. **Тестирует функциональность** и формирует интерес к заказу

### Для администраторов

1. **Админ заходит** в админ-панель: `/admin`
2. **Просматривает контакты** оставленные пользователями
3. **Фильтрует по категориям** и датам
4. **Экспортирует данные** для дальнейшей работы
5. **Анализирует интересы** потенциальных клиентов

## 🔧 Технические детали

### Архитектура
```
Telegram Bot → Express Server → Mini App Frontend
                    ↓
               Contact Storage
                    ↓
               Admin Panel
```

### Используемые технологии
- **Backend**: Node.js, Express, Telegraf
- **Frontend**: HTML, CSS, JavaScript
- **Хранение**: In-memory (для демо)
- **Деплой**: Vercel
- **Интеграция**: Telegram Web Apps SDK

### API endpoints
- `GET /health` - Проверка состояния
- `POST /api/contacts` - Сохранение контакта
- `GET /api/contacts` - Получение контактов
- `GET /api/contacts/category/:category` - Поиск по категории
- `DELETE /api/contacts/:id` - Удаление контакта
